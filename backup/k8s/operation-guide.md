# K8s Operation Guide

## 개요

Kubernetes 운영 기준은 중복 방지를 위해 [K8s Installation](./installation.md) 문서로 통합했습니다.
이 문서는 기존 링크 호환성을 위한 안내 문서입니다.

## 기준 문서

- 구축 절차: [K8s Installation](./installation.md)
- 설치 직후 검증: [K8s Installation](./installation.md#설치-검증)
- 일일/주간/월간 점검: [K8s Installation](./installation.md#설치-직후-운영-기준)
- 유지보수/업그레이드/복구 시작점: [K8s Installation](./installation.md#유지보수-및-변경-표준)
- 장애 패턴별 조치: [K8s Troubleshooting](./troubleshooting.md)

## 운영 시 주의사항

- 새 절차를 추가할 때는 `installation.md`를 기준 문서로 먼저 갱신합니다.
- 반복되는 운영 명령을 별도 문서에 다시 복제하지 않습니다.
